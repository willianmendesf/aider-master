# Fluxo Obrigatório
1. Descobrir contexto.
2. Criar inventário.
3. Mapear dependências.
4. Planejar implementação.
5. Validar plano.
6. Implementar.
7. Revisar.
8. Testar.
9. Auditar.

# Regras
Nunca: Ler → Implementar
Sempre: Ler → Entender → Planejar → Validar → Implementar
